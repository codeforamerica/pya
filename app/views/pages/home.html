A free state filing service for taxpayers using IRS Direct File

TESTTTT

FileYourStateTaxes connects with IRS Direct File to help you complete your state tax return for free.

This site will open at the start of the 2025 tax filing season and remain open until October 15, 2025 for you to file your state tax returns.

In 2025, this service will support filing state tax returns in Arizona, Idaho, North Carolina, New Jersey, and Maryland.

Already started your state return and need to complete it? Sign in here.

  1. First, file and submit your federal return with IRS Direct File before using this service.
  2. Follow the link from Direct File to FileYourStateTaxes.
  3. Answer a few questions to make sure this is the right service for you. Learn more about who is eligible for this tax year.
  4. Transfer your federal return to fill in most of your state return.
  5. Answer a few additional questions to help maximize your state tax refund.
  6. E-file your return for free!


To learn more, check out the FAQs.
Looking for your 2023 Arizona or New York State Tax Return? Sign in to download a copy of your state return.

FileYourStateTaxes is built by Code for America in partnership with state tax agencies to integrate with IRS Direct File.
